# QA_API

SDC - System Design Capstone

The backend server and database for a multiple component application.


Technologies used:
- Node.js
- Express
- Postgres
- Frisby
- Jest
- k6
- Loader.io
- New Relic
- NGINX
- PGTune

Installation:
- run 'npm install' to installdependencies.
- run 'npm run start' to start the server.
- Navigate to localhost:3000
